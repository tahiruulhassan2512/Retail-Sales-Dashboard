# Retail-Sales-Dashboard
📌 Project Overview
This project focuses on analyzing retail sales data using Power BI to understand business performance and generate meaningful insights.

🎯 Objectives
Analyze overall sales and profit
Identify top-performing products
Understand regional performance
Track monthly sales trends
Create an interactive dashboard

📁 Dataset
The dataset contains information about:
Sales
Profit
Quantity
Category & Sub-category
Region & State


📊 Dashboard Features
Total Sales, Profit, Orders, and Quantity cards
Sales by Category (Bar Chart)
Sales by Region (Pie Chart)
Monthly Sales Trend (Line Chart)
Top 10 Products by Sales
Interactive Slicers (Region & Segment)

🔑 Key Insights
Technology category has the highest sales
West region contributes the most revenue
Sales increase towards the end of the year
Top products significantly impact total sales

🛠 Tools Used
Power BI
Excel (for dataset)

Conclusion
The dashboard helps in analyzing sales performance and supports better business decisions.
It also provides clear insights into trends, top products, and regional performance.
